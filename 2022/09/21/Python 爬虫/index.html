Python 爬虫

一、什么是爬虫

1. 爬虫Spider的概念

爬虫的数据来源于网络，网络中的数据可以是有web服务器（Nginx/Apache）、数据库服务器（MySql、Redis）、索引库（ElasticSearch）、大数据（Hbase/Hive）、视频/图片库（FTP）、云存储（OOS）提供的。

2. Python 爬虫技术

主要有多（单）线程/进程、网络请求库、数据解析、数据存储、任务调度等相关技术。

可以完成接口测试、功能性测试、性能测试和集成测试。

相关的库：

  • urllib
  • requests
  • selenium # UI自动化测试、动态js渲染
  • appuim # 手机App的爬虫或UI测试

数据解析：

  • re 正则
  • xpath
  • bs4
  • json

数据存储：

  • pymysql
  • MongoDB
  • elasticsearch

多任务库：

  • 多线程（threading）、线程队列（queue）
  • 协程（asynio、gevent/eventlet）

爬虫框架：

  • scrapy
  • scrapy-redis 分布式（多机爬虫）

3. 反爬虫策略

  • UA（User-Agent）策略
  • 登陆限制（Cookie）策略
  • 请求频次（IP代理）策略
  • 验证码（图片-云打码，文字或物件图片选择，滑块）
  • 动态js（Selenium/Splash/api接口）策略

二、requests库

1. 核心的函数

  • requests.request() 所有请求方法的基本方法

    以下是request()方法的参数说明

    • method：str 指定请求方法，GET,POST,PUT,DELETE

    • URL：str 请求的资源接口（API），在restful规范中既是URI（统一资源标识符）

    • params：dict ，用于 GET 请求的查询参数

    • data：dict，用于 POST/PUT/DELETE 请求的表带参数（From Data）

    • json：dict，用于上传json数据的参数，封装到body（请求体）中。请求头的Content-Type默认设置为 application/ json

    • files：dict，结构{‘name’:file-like-object | tuple},如果是 tuple，则有三种情况：

      • (‘filename’:file-like-object)
      • (‘filename’:file-like-object,content_type)
      • (‘filename’:file-like-object,content_type，custome-headers)

      指定files用于上传文件，一般使用post请求，默认请求头的Content-Type为multipart/form-data类型。

    • headers/cookies : dict

    • proxies : dict, 设置代理

    • auth：tuple，用于授权的用户名和口令，形式（’user’,’passwd’）

  • requests.get() 查询数据

    • url
    • params
    • json
    • headers/cookies/auth

  • requests.post() 上传/添加数据

    • url
    • data
    • json
    • files
    • headers/cookies/auth

  • requests.put() 修改或更新数据

  • requests.delete() 删除数据

3. requests.Response

以上的请求方法返回的对象类型是Response，对象常用的属性如下：

  • status_code 响应状态码

  • url 请求的url

  • headers：dict，响应的头，相对于urllib的响应对象的getheaders()，但不包含cookie

  • cookies：可迭代的对象，元素是Cookie类对象（name，value，path）

  • text：响应的文本信息

  • content：响应的字节数据

  • encoding：响应数据的编码字符集，如utf-8，gbk，gb2312

  • json()：如果响应的数据类型为 application/json，则将响应的数据进行反序列化成python的list或dict对象
